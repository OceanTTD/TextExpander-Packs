TextExpander Packs

TextExpander snippet packs for developers

These snippet groups are perfect for developers. Each group is easy to add to TextExpander by using the "Add Group from URL" feature, just copy your desired group from the links below.

Packs available:

  • Apache
  • CSS, Sass
  • Git, SVN (Subversion)
  • Linux, Ubuntu, CentOS
  • MySQL, SQL
  • PHP
  • Ruby
  • Shell commands
  • Drush (Drupal)
